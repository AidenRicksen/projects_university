# University Projects — Aiden Ricksen

A collection of university projects completed at **Tilburg University** as part of the BSc Cognitive Science and Artificial Intelligence programme. The projects span programming, machine learning, natural language processing, and simulation.

---

## Projects

### [Castle Defender](./castle-defender/)

**Course:** Programming with C/C++ — Fall 2025/2026

A turn-based tower defense game built in C++17 with an FLTK graphical user interface. The player places towers on a grid to defend a castle against waves of enemies. Enemies use an adaptive AI that selects spawn positions via a heatmap and scales difficulty dynamically based on player performance. The codebase follows a clean MVC architecture, separating game logic from the GUI layer.

**Technologies:** C++17, FLTK, OOP, MVC architecture

---

### [Binary Classification — Social Media](./binary-classification/)

**Course:** Machine Learning / Data Mining

A binary classification project applied to social media data. Covers the full machine learning pipeline: data preprocessing, feature engineering, model selection, hyperparameter tuning, and evaluation of classifiers for predicting social media outcomes.

**Technologies:** Python, scikit-learn, pandas, data analysis

---

### [Computational Linguistics](./computational-linguistics/)

**Course:** Computational Linguistics

A project exploring computational approaches to natural language. Involves implementing and evaluating NLP techniques for linguistic analysis tasks.

**Technologies:** Python, NLP, linguistic analysis

---

### [Turtle Age Regression](./turtle-age-regression/)

**Course:** Machine Learning / Data Mining

A regression project using biological measurements to predict the age of turtles. Demonstrates end-to-end regression modelling including feature selection, model training, and performance evaluation.

**Technologies:** Python, scikit-learn, regression modelling

---

### [Vaccination and Mutation Simulation](./vaccination-mutation-simulation/)

**Course:** Introduction to Programming / Simulation

An agent-based simulation modelling the spread, mutation, and vaccination effects of a disease within a population. Visualises how vaccination rates and mutation probability affect outbreak dynamics over time.

**Technologies:** Python, simulation, agent-based modelling
